# weathergraphic

## Install Flask
*  python3 -m venv .venv
* . .venv/bin/activate
* pip install Flask

## How to run the Program
* From the working directory export FLASK_APP=weather.py
* flask run

## Installing packages
ex: python3 -m pip install requests

## URL
http://127.0.0.1:5000/observations
